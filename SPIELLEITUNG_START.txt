L&B PRESSEARCHIV 2014 - START

1. start_game.cmd doppelt anklicken.
2. Das schwarze Fenster geöffnet lassen. Automatisch öffnet sich die lokale Seite "Spielleitung".
3. Den dort bzw. im schwarzen Fenster angezeigten Link "Für Teilnehmende" an alle Gruppen geben.
4. Die Teilnehmenden dürfen ihre Seiten bereits öffnen; die Spieluhr steht dabei auf BEREIT.
5. Erst zum offiziellen Beginn auf "SPIELTAG JETZT STARTEN" klicken. Dann laufen die fünf Stunden verbindlich.
6. Alle Geräte müssen sich im selben lokalen Netzwerk/WLAN befinden.
7. Falls Windows beim ersten Start fragt, den Zugriff im privaten Netzwerk zulassen.

Vor einem neuen Durchlauf:
1. Spielserver schließen.
2. reset_game.cmd ausführen.
3. start_game.cmd erneut starten.

Zugänge:
Einführungsrätsel: bildet gruppen
Gruppe A: personen -> Ergebnis 8
Gruppe B: orte -> Ergebnis 4
Gruppe C: jahre -> Ergebnis 5
Gruppe D: transaktionen -> Ergebnis 2
Gruppe V: regeln -> Code 0764
Segment E: reihenfolge -> finaler Code 64-07 (6407 wird ebenfalls akzeptiert)

Ratensperre:
Drei falsche Ergebniseingaben innerhalb von fünf Minuten sperren die jeweilige Sitzung für zehn Minuten.

Zeitliche Sicherheitsnetze ab offiziellem Start:
- Gruppen A-D: Spuren bei Minute 30, 55 und 80; externe Rekonstruktion ab Minute 90.
- Gruppe V: Spuren bei Minute 95, 105 und 115; externe Rekonstruktion ab Minute 120.
- Segment E: Spuren bei Minute 145, 175 und 200; externe Rekonstruktion ab Minute 210.
- Das erreichte Finale wird automatisch auf allen noch geöffneten Gruppenansichten angezeigt.

Technischer Hinweis:
Der Spielstand wird zentral in game-state.json gespeichert. Die Gruppenauswahl ist pro Browsersitzung bindend. Ein absichtliches Umgehen durch Inkognito-Modus oder ein anderes Gerät kann technisch nicht vollständig verhindert werden.
